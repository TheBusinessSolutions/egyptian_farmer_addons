Overview

Secure your sales process by controlling who can modify product prices in Sales Orders. With this module, only authorized users can adjust unit prices based on pricelists, while others are restricted. Any unauthorized change attempt instantly triggers a warning ensuring transparency, control, and pricing integrity.

Features

Configuration:

  • Go to Settings > Manage Price Change.
  • Open the user profile and navigate to Other Extra Rights.
  • Check the option ‘Manage Sales Price Change’ to grant access.
  • Note: Make sure Developer Mode is enabled to view this option
